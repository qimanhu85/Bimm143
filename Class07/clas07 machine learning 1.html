K-means clustering

The main function in “base” R for K-means clustering is called kmeans()

Before we go too deep let’s make up some “simple” data that we can cluster. To do this we can use the rnorm() function:

hist(rnorm(1000,mean=3, sd=))
x <- c(rnorm(30,-3),rnorm(30,+3))
z <- cbind(x=x,y=rev(x))
#rev(x)
plot(z)

Now we can run kmeans() on this input z and see what the results look like.

km <- kmeans(z, centers=2)

Hierarchical Clustering

The main function in “base” R for Hierarchical Clustering is called hclust()

This function does not take your raw data for clustring. You must first build a “distance matrix” from your data and pass this as input to hclust()

d <- dist(z)
hc <- hclust(d)
hc

Call:
hclust(d = d)

Cluster method   : complete 
Distance         : euclidean 
Number of objects: 60

There is a bespokeplot() method for hclust() result objects.

plot(hc)
abline(h=8,col="red")

Once we have our hclust object (our tree of cluster dendrogram) we can cut the tree to reveal the clustering pattern.

cutree(hc, h=8)
